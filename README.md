# Nausicaa Consult

Landing page for Nausicaa Consult — business capability and tendering consultancy based in Broome, WA, serving the Kimberley, Pilbara and Gascoyne regions.

## Structure

- `index.html` — single-page site (hero, about, services, regions, case study request, contact)
- `css/styles.css` — styling
- `js/main.js` — smooth scroll + footer year

## Before going live

- Swap placeholder hero/about visuals for real photography/logo when available.

## Local preview

Open `index.html` directly in a browser, or run a simple static server:

```
python3 -m http.server 8000
```
